# 第3章 深度学习与神经网络

## 3.1 深度学习概述

深度学习是机器学习的一个分支，使用多层神经网络来学习数据的层次化表示。它的核心思想是：**通过组合简单特征来学习复杂概念。**

深度学习的"深度"指的是网络中的层数。浅层网络可能只有1-2个隐藏层，而深度网络可以有几十甚至上百层。

## 3.2 神经网络的数学基础

### 神经元模型
一个人工神经元接收多个输入x₁, x₂, ..., xₙ，每个输入有对应的权重w₁, w₂, ..., wₙ，加上一个偏置b：

z = Σ wᵢxᵢ + b

然后通过激活函数f(z)得到输出：
a = f(z)

### 常用激活函数
- **Sigmoid**：σ(x) = 1/(1+e⁻ˣ)，输出范围(0,1)，适合二分类输出层
- **Tanh**：tanh(x)，输出范围(-1,1)，零中心化
- **ReLU**：max(0, x)，计算简单，缓解梯度消失，最常用
- **Leaky ReLU**：max(αx, x)，解决ReLU的"神经元死亡"问题
- **Softmax**：eˣⁱ/Σeˣʲ，将输出转化为概率分布，多分类输出层

### 损失函数
- **交叉熵损失（Cross-Entropy Loss）**：分类任务的标准损失函数
- **均方误差（MSE）**：回归任务的常用损失函数
- **Hinge Loss、Focal Loss** 等变体

### 梯度下降与反向传播
**梯度下降**是最基本的优化算法：
- Batch Gradient Descent：使用全部数据计算梯度
- Stochastic Gradient Descent (SGD)：每次使用一个样本
- Mini-batch SGD：使用一个小批量样本（最常用）

**反向传播算法**通过链式法则高效计算每层的梯度：
1. 前向传播：计算预测值和损失
2. 反向传播：从输出层向输入层逐层计算梯度
3. 参数更新：使用优化器更新权重

## 3.3 经典深度学习架构

### 卷积神经网络（CNN）
专门为处理网格结构数据（如图像）设计。

**核心组件**：
- **卷积层（Convolutional Layer）**：使用可学习的卷积核在输入上滑动，提取局部特征
- **池化层（Pooling Layer）**：下采样，减少参数量
  - 最大池化（Max Pooling）、平均池化（Average Pooling）
- **全连接层（Fully Connected Layer）**：将提取的特征映射到输出

**经典架构**：
- LeNet-5（1998）：手写数字识别
- AlexNet（2012）：ImageNet冠军，深度学习复兴的标志
- VGGNet（2014）：使用小卷积核堆叠
- GoogLeNet/Inception（2014）：Inception模块，多尺度卷积
- ResNet（2015）：残差连接，解决了深层网络退化问题
- DenseNet（2017）：密集连接，每层与所有前层连接

### 循环神经网络（RNN）
处理序列数据的经典架构。

**核心思想**：隐藏状态hₜ不仅依赖当前输入xₜ，还依赖上一时刻的隐藏状态hₜ₋₁。

**问题**：梯度消失/爆炸，导致难以学习长距离依赖

**改进变体**：
- **LSTM（长短期记忆网络）**：通过遗忘门、输入门、输出门控制信息流动
- **GRU（门控循环单元）**：LSTM的简化版，参数更少

**应用**：文本生成、机器翻译、语音识别、时间序列预测

### Transformer（2017）
《Attention is All You Need》论文提出的革命性架构，完全基于注意力机制，抛弃了RNN的循环结构。

**核心组件**：
- **Self-Attention（自注意力）**：Q, K, V都来自同一个输入序列
- **Multi-Head Attention（多头注意力）**：多个注意力头并行计算
- **Positional Encoding（位置编码）**：为序列添加位置信息
- **Feed-Forward Network（前馈网络）**：逐位置的全连接层
- **Layer Normalization & Residual Connection**

**注意力计算公式**：
Attention(Q, K, V) = softmax(QKᵀ/√dₖ)V

**优势**：（1）并行计算，训练效率高；（2）长距离依赖建模能力强；（3）可扩展到超大规模

## 3.4 生成模型

### 变分自编码器（VAE）
学习数据的潜在分布，通过编码器-解码器结构生成新样本。

### 生成对抗网络（GAN）
由生成器G和判别器D组成。生成器试图生成逼真的假样本，判别器试图区分真假样本。两者通过对抗训练共同进化。

**经典GAN变体**：DCGAN、WGAN、StyleGAN、CycleGAN、Pix2Pix

### 扩散模型（Diffusion Models）
通过学习逆扩散过程来生成数据。前向过程逐步向数据加噪，逆向过程从噪声中恢复数据。

**代表模型**：DDPM、Stable Diffusion、DALL-E 2/3、Midjourney

## 3.5 大语言模型（LLM）

### GPT系列
- GPT-1（2018）：1.17亿参数，预训练+微调范式
- GPT-2（2019）：15亿参数，zero-shot能力
- GPT-3（2020）：1750亿参数，few-shot/in-context learning
- GPT-4（2023）：多模态，推理能力大幅提升

### 关键技术
- **预训练+微调**：先在大规模语料上预训练，再在下游任务微调
- **RLHF**（人类反馈强化学习）：让模型对齐人类偏好
- **Prompt Engineering**：通过设计提示词引导模型输出
- **RAG**（检索增强生成）：结合外部知识库减少幻觉
- **Agent**：让LLM具备工具使用和自主决策能力

## 3.6 训练技巧

### 参数初始化
- Xavier/Glorot初始化：适合Sigmoid/Tanh
- He初始化：适合ReLU

### 正则化技术
- Dropout：训练时随机丢弃一部分神经元
- Batch Normalization：对每层输入进行归一化
- Layer Normalization：在特征维度归一化（Transformer中常用）
- Weight Decay：L2正则化

### 优化器
- SGD + Momentum
- Adam（最常用）：自适应学习率
- AdamW：Adam + Weight Decay解耦

### 学习率调度
- Step Decay、Cosine Annealing、Warmup + Decay

## 本章小结

本章介绍了深度学习的核心概念和主流架构。从基础神经元、激活函数到CNN、RNN、Transformer等经典架构，再到生成模型和大语言模型。理解这些架构的设计思想和适用场景，是开发AI应用的基础。
